Chester - React Portfolio Documentation

Introduction

Chester is a React and Bootstrap 4 based personal portfolio website template. This has been built with React, ES6+ etc. This is created especially for those who want to build up his portfolio site (Developer, Designer).

Setup Development Environment

In order to kickstart the development of your web app with Chester, the very first thing you need to do is to setup a react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site http://nodejs.org/

Preparation

You'll need to install Node.js >=v10.0+ (Recommended Version) (NPM comes along with it) and serve (a NPM package)

Use command line tool i.e. powershell or terminal and navigate to the project(Chester) root. This is powershell in windows and terminal on mac/linux.

Let's see how the setup works:

Use terminal and navigate to the project root.

Then run : npm install

Then run : npm run start

Now in the browser go localhost:3000

Fonts

By default, the template loads Nunito font from Google Web Font Services, you can change the font with the one that suits you best.

You will find the font code in the "scss/basic/_typography.scss


    @import url('https://fonts.googleapis.com/css?family=Nunito:300,400,400i,600,700,800&display=swap');
